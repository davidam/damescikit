* What?

That's a package to run tests about scikit library of Python.
For example, create models, vectorizer


* Installing
** Create a python environment
#+BEGIN_SRC
$ mkdir venv3.14
$ python3 -m venv venv3.14
$ cd venv3.14
$ source bin/activate  
#+END_SRC

** Install the python package
#+BEGIN_SRC  
$ pip3 install damescikit
#+END_SRC

** Access to the path of the package
#+BEGIN_SRC  
$ cd lib/python3.14/site-packages/damescikit
#+END_SRC

* Check Test

** In the path:

#+BEGIN_SRC  
$ cd damescikit
#+END_SRC

** Execute all tests:

#+BEGIN_SRC
$ pytest tests
#+END_SRC

** Execute one file:

#+BEGIN_SRC
$ pytest tests/test_models.py
#+END_SRC

** Execute one single test in a file

#+BEGIN_SRC
$ pytest tests/test_models.py::TddInPythonExample::test_models_lda
#+END_SRC
